# SnailColony

Monorepo scaffold for the SnailColony proof of concept game.

## Development

```bash
pnpm install
pnpm dev
```

The application uses Docker for dependencies:

```bash
pnpm compose:up
```

Access the client at http://localhost/ and connect to a server via WebSocket URL.
